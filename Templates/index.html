Purpose

The purpose of this project is to predict student performance based on various attributes. By analyzing factors such as age, gender, parental education level, attendance rate, hours of study, extracurricular activities, and more, we aim to provide insights into the factors that influence a student's academic success.

Data Source

The dataset used in this application contains synthetic data generated for Indian student records. While the data is fictional, it is based on realistic distributions and characteristics commonly found in student populations.

Features

The features used in this model include age, gender, parental education level, location, attendance rate, hours of study, extracurricular activities, previous academic performance, semester gap, area of interest, future goals, and more. These attributes have been carefully selected based on their potential impact on student performance.

How to Use

To predict student Behavior, simply navigate to the "Student Performance Prediction" section and input the relevant student attributes. Upon clicking the "Predict" button, the application will display the predicted CGPA for the student based on the provided information.
